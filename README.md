# Retail-Store-SQL-Analysis
SQL-based retail store database project featuring database design, data modeling, joins, aggregations, and business analysis queries.

## Project Overview

This project demonstrates the design and analysis of a Retail Store Database using SQL. The database manages customers, products, orders, order items, and payments while enabling business-focused data analysis.

## Database Structure

### Tables
- Customers
- Products
- Orders
- Order Items
- Payments
